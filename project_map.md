# خريطة المشروع والوضع الحالي (Project Map)

## 1. الفكرة العامة للمشروع والهدف الاستثماري:
* **منصة وصف (Wasf):** منصة Micro-SaaS ذكية لأتمتة وتوليد وصف المنتجات والمحتوى التسويقي للمتاجر الإلكترونية السعودية (مثل سلة وزد) باستخدام Gemini API.
* **الهدف:** حل مشكلة صياغة المحتوى الجاف للمنتجات، وتحويله إلى نصوص بيعية جذابة متوافقة مع محركات البحث (SEO) ومكتوبة بـ "اللهجة البيضاء السعودية المحلية" التي تلامس ثقافة وعاطفة المستهلك المحلي لرفع نسب المبيعات والتحويلات.
